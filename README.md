# Flowchart

> 단일 HTML 파일로 동작하는 가벼운 업무용 흐름도 도구

**외부 라이브러리 0개 · 설치 불필요 · 오프라인 실행 · 68KB**

회사 보안 환경(VDI)에서도 브라우저로 HTML 파일 하나만 열면 바로 사용할 수 있는 **미니멀 Flow Diagram 도구**입니다.

draw.io가 너무 무겁거나, 보안 정책으로 외부 서비스를 쓸 수 없는 환경에서 **빠르게 업무 흐름을 정리하고 PNG로 공유**하는 것이 목적입니다.

---

## ✨ 주요 기능

### 도형 & 메모

- **프로세스**(둥근 사각형), **판단**(마름모), **시작/종료**(캡슐형) 3종 도형
- **자유 텍스트 메모** — 흐름도 옆에 설명, 예외 조건, 담당자 메모 작성
- 도형/메모 **사이즈 조절** — 8방향 핸들 드래그
- **더블클릭** → 텍스트 편집 (Enter=확정, Shift+Enter=줄바꿈)
- **도형 배경 색상** 6종 + **글자 색상** 변경

### 연결선

- 노드 핸들에서 **드래그하여 연결** — 출발/도착 방향을 기억
- **직선 / 꺾인선(직교)** 개별 전환 — 화살표 선택 후 툴바 또는 우클릭
- **연결선 색상** 5종 (기본, 블루, 레드, 그린, 오렌지)
- 마름모 Y/N 분기 시 색상으로 구분 가능

### 선택 & 이동

- **클릭** → 단일 선택
- **Shift + 클릭** → 멀티 선택 (토글)
- **빈 영역 드래그** → 범위 선택
- **드래그** → 선택된 도형/메모 그룹 이동
- **화살표 키** → 1px 이동 / **Shift + 화살표** → 10px 이동
- **Space + 드래그** → 캔버스 전체 이동

### 정렬

복수 선택(2개 이상) 시 툴바에 정렬 버튼 표시:

| 버튼 | 기능 |
|:---:|---|
| ◧ | 왼쪽 정렬 |
| ⫐ | 가로 중앙 정렬 |
| ◨ | 오른쪽 정렬 |
| ⊤ | 위 정렬 |
| ⊶ | 세로 중앙 정렬 |
| ⊥ | 아래 정렬 |
| ⇔ | 가로 균등 분배 |
| ⇕ | 세로 균등 분배 |

### 복사 & 붙여넣기

- **Ctrl + C / V** — 단일 또는 멀티 복사·붙여넣기
- 복수 선택 시 **도형 사이의 연결선도 함께 복사**
- 연속 붙여넣기 시 계단식 배치

### 저장

- **Ctrl + S** → 파일명 입력 후 JSON 저장 (이전 파일명 기억)
- **JSON 불러오기** → 작업 이어서 하기
- **PNG 저장** → 내용 영역만 자동 크롭, 2배 해상도
- **localStorage 자동저장** → 브라우저 닫았다 열어도 마지막 상태 복원
- **브라우저 닫기/새로고침 방지** — 실수로 닫아도 경고 표시

### 기타

- **Ctrl + Z / Y** — Undo / Redo (최대 100단계)
- **Delete / Backspace** — 선택 항목 삭제
- **캔버스 크기** 직접 입력 가능 (기본 3200×2000)
- **우클릭 메뉴** — 상황별 메뉴 제공

## 🚀 사용 방법

1. `index.html` 파일을 다운로드합니다.
2. 브라우저에서 더블클릭하여 엽니다.
3. 끝.

인터넷 연결, 서버, 설치 — 아무것도 필요 없습니다.

## 💡 만든 배경

PPT로는 담기지 않는 복잡한 업무 프로세스를 정리하고 싶었지만, 회사 보안 환경(VDI)에서 draw.io 같은 외부 서비스를 사용할 수 없었습니다.

**"HTML 파일 하나만 열어서 흐름도를 그리고 PNG로 저장할 수 있는 도구"** 를 목표로, Claude AI와 대화하며 기획 → 구현 → 반복 개선 과정을 거쳐 완성했습니다.
